HTML5 is not a version - it is the latest evolution of the standard that defines HTML. Also includes HTML5 and friends (a larger set of technologies that allows building of more diverse and powerful Web sites and applications.)

There is a 'HTML standard' that is how browsers build the HTML based off the standard.

You can find HTML standard here: html.spec.whatwg.org

Block Elements

Examples: Paragraph, headers

Inline elements

For example: Anchor tags, images, they share the line - the block elements take up their own line. You can change this in CSS later. You can change this with Span and Div

Div = Content Division Element - can be used to group together content. You can then treat the content similarly.
