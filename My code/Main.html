Divs and Spans

Divs

Used to group sections together and when applying CSS can apply the same formatting to that 'block'

Creates a 'block' (all the way across the page) rather than 'inline' text - e.g. for putting a border around a section of text

Spans

Serves a similar purpose to Divs - a generic container but it is an 'inline' element

Wrap around any text or content that we want to style with a certain set of CSS on

You put a span class = "" and this allows you to surround it with customer formatting

Example - could go style the word 'Google' differently using CSS
